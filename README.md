# QuickHire - Job Hunt Platform

QuickHire is a modern, high-performance job hunting platform built with **Next.js 15**, **React 19**, and **Tailwind CSS 4**. It features a premium, human-centric design with smooth animations and a responsive layout.

## 🚀 Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Library**: [React 19+](https://react.dev/)
- **Styling**: [Tailwind CSS 4+](https://tailwindcss.com/)
- **Fonts**: Clash Display (Headings), Epilogue (Body)
- **Icons**: Custom SVG icons and Optimized PNGs

## ✨ Features (Currently Implemented)

- **Premium Hero Section**: Modern design with responsive layout and custom typography.
- **Dynamic Companies Section**: Grayscale-to-color hover effects for partner logos with standardized sizing.
- **Category Explorer**: Functional category grid with custom hover states and brand-colored highlights (e.g., Marketing).
- **Responsive Footer**: Multi-column layout optimized for both desktop and mobile views.
- **SEO Optimized**: Built with semantic HTML and performance best practices.

## 🛠️ Getting Started

### Prerequisites

- Node.js 20.x 
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/esmotara-codec/job-hunt-client.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
